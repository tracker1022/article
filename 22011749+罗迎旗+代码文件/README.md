# 主从博弈模型构建的市场机制

本项目实现了一个基于主从博弈模型的市场机制，用于模拟电力市场中的多主体交互。

## 项目结构

- `models.py`: 包含各个市场参与者的模型定义
- `main.py`: 主程序文件，实现市场机制的运行逻辑
- `requirements.txt`: 项目依赖文件

## 主要功能

1. 模拟传统发电机和可再生能源发电机的发电行为
2. 模拟用户的用电行为
3. 模拟电动汽车的充放电行为
4. 实现虚拟电厂的优化决策
5. 实现主网的电价调整机制

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行方法

```bash
python main.py
```

## 模型说明

### 市场参与者

1. 传统发电机
   - 发电成本为二次函数
   - 有最大和最小发电容量限制

2. 可再生能源发电机
   - 发电成本为一次函数
   - 有最大和最小发电容量限制

3. 电动汽车
   - 考虑电池容量动态变化
   - 充放电行为影响电池状态

4. 用户
   - 使用对数效用函数
   - 有最大和最小用电需求限制

### 市场机制

1. 主网层
   - 根据供需关系调整电价
   - 考虑影响因子

2. 虚拟电厂层
   - 优化购售电决策
   - 平衡内部供需

3. 用户层
   - 根据电价调整用电行为
   - 最大化自身效用
